# 인스타그램 카드뉴스 자동화 파이프라인 기획서

## 개요

서포터즈 활동을 위한 인스타그램 카드뉴스 제작을 자동화하는 파이프라인.
원문 텍스트를 입력하면 Claude가 슬라이드 텍스트를 생성하고, HTML 렌더링 후 Playwright로 PNG를 캡처한다.

---

## 파이프라인 흐름

```
원문 텍스트 입력 (CLI)
        ↓
Claude API — 슬라이드별 텍스트 요약 생성 + 커버 훅 문구 자동 생성
        ↓
이미지 폴더 스캔 — 파일명 번호로 슬라이드 매핑 (slide2.jpg → 2장)
        ↓
이미지 유무에 따라 레이아웃 타입 자동 선택
        ↓
테마 CSS + 슬라이드 HTML 파일 생성 (장당 1개)
        ↓
Playwright로 1080×1350px PNG 캡처
        ↓
output/ 폴더에 저장 → 수동으로 인스타그램 업로드
```

---

## CLI 실행 방식

```bash
python main.py \
  --theme tech \
  --topic "Claude 3.5 Sonnet 활용 후기" \
  --text "원문 텍스트 전문..." \
  --images ./images/week3/
```

| 인자 | 설명 |
|---|---|
| `--theme` | `tech` / `activity` / `trend` 중 선택 |
| `--topic` | 카드뉴스 제목/주제 |
| `--text` | 슬라이드 생성에 사용할 원문 텍스트 |
| `--images` | 이미지 폴더 경로 (없으면 text-only로 처리) |

## 슬라이드 구성

| 장 | 역할 | 이미지 | 레이아웃 옵션 | 비고 |
|---|---|---|---|---|
| 1장 | 커버 | 선택 | 배경 이미지형 / 컬러 배경형 | 고정 디자인 |
| 2~4장 | 본문 | 선택적 | `image-top` / `image-split` / `image-blur-bg` / `text-only` | Claude 자동 선택 |
| 5장 | 정리 | 선택적 | 불릿형 / 블러 배경 + 불릿형 | 핵심 요점 나열 |
| 6장 | CTA | 없음 | 로고 + 멘트 + 해시태그 | 고정 디자인 |

### 본문 레이아웃 타입 정의

| 타입 | 조건 | 구조 |
|---|---|---|
| `image-top` | 이미지 있음 | 상단 45% 이미지 + 하단 텍스트 |
| `image-split` | 이미지 있음 (가로형) | 좌우 분할 (이미지 40% / 텍스트 60%) |
| `image-blur-bg` | 이미지 있음 (배경용) | 이미지 블러 전체 배경 + 텍스트 오버레이 |
| `text-only` | 이미지 없음 | 텍스트 전용 |

레이아웃 타입은 이미지 유무와 비율을 Claude가 감지하여 자동 선택한다.

## 이미지 파일명 규칙

이미지 폴더 내 파일명에 슬라이드 번호를 포함한다.

```
images/week3/
├── slide2.jpg   # 2장 본문에 사용
├── slide3.png   # 3장 본문에 사용
└── slide5.jpg   # 5장 정리에 사용
```

- 번호가 없는 슬라이드는 `text-only` 레이아웃으로 처리
- 지원 확장자: `.jpg`, `.jpeg`, `.png`, `.webp`

## Claude API 역할 정의

### 슬라이드 텍스트 생성 (`generator.py`)

Claude에게 전달하는 프롬프트 구조:

```
[시스템]
당신은 인스타그램 카드뉴스 콘텐츠 작가입니다.
입력된 원문을 바탕으로 5~6장 분량의 슬라이드 텍스트를 JSON으로 생성하세요.

[출력 형식]
{
  "cover": {
    "hook": "커버 훅 문구 (15자 이내, 호기심 유발)",
    "subtitle": "부제목"
  },
  "slides": [
    {
      "index": 2,
      "title": "슬라이드 제목",
      "body": "본문 내용 (3~4줄 이내)"
    }
  ],
  "summary": {
    "points": ["요점 1", "요점 2", "요점 3"]
  }
}
```

- 커버 훅 문구는 Claude가 자동 생성 (사용자가 이후 직접 수정 가능)
- JSON 출력만 반환, 마크다운 코드블록 없이

## 개발 순서 (권장)

1. 프로젝트 구조 및 CLI 뼈대 구성
2. Claude API 연동 및 슬라이드 텍스트 생성 (`generator.py`)
3. 테마 CSS 파일 3종 작성
4. HTML 템플릿 4종 작성 (커버 / 본문 / 정리 / CTA)
5. 레이아웃 자동 선택 로직 구현
6. Playwright 캡처 구현 (`capture.py`)
7. 전체 파이프라인 통합 테스트
8. GitHub 레포 정리 및 README 작성
